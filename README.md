# YouTube Gallery (GitHub Pages)

## Apa ini
- `gallery.html` = landing page gallery yang baca `data.json`
- `admin.html` = local admin page untuk create/edit/import/export `data.json`

## Cara guna (GitHub Pages)
1. Upload semua fail dalam folder ini ke repo GitHub.
2. Enable GitHub Pages (Settings → Pages).
3. Buka `gallery.html` (atau set sebagai root/index).

## Cara guna (Admin / Local)
1. Buka `admin.html` secara local (double click).
2. Tambah/Edit/Delete video.
3. Export → akan download `data.json`.
4. Replace `data.json` dalam repo GitHub dengan file baru.
